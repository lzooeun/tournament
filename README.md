# 🏆 TaekTube Tournament (E-Sports Management Platform)

A comprehensive full-stack web platform and automated Discord bot designed to manage, operate, and balance a custom League of Legends e-sports tournament. 

## 🛠 Tech Stack
- **Web Backend:** Python, Django
- **Bot Development:** discord.py
- **Frontend:** HTML5, CSS3, JavaScript
- **Core Logic:** Custom Matchmaking & Tier Balancing Algorithm

## 🙋‍♀️ My Role & Contributions (Jooeun Lee)
As the lead developer and co-planner of the tournament, I built the entire digital infrastructure from scratch, handling both the technical development and live operations:
- **Full-Stack Web Platform:** Developed a Django-based web dashboard to display real-time match schedules, player rosters, and official tournament rules.
- **Discord Bot Automation:** Built a 24/7 custom Discord bot to automate the entire participant journey—from team sign-ups and self-introductions to match result submissions.
- **Matchmaking Algorithm:** Designed and implemented a custom mathematical algorithm that calculates player tiers to generate perfectly balanced teams and tournament brackets.
- **UI/UX & Operations:** Led the visual design of the web platform and managed the live operational data, including real-time player tier adjustments to ensure fair play.

## ✨ Key Features
- **Automated Tournament Operations:** Streamlined participant registration and administrative tasks via intuitive Discord commands.
- **Fair Team Generation:** Custom tier-balancing algorithm that automatically evaluates player stats and forms balanced rosters.
- **Real-Time Web Dashboard:** A centralized, responsive web hub for players and viewers to track the tournament's progress and brackets live.
